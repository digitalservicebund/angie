# Typography

Typografie kann in ihrer Größe, Farbe und ihrem Schriftschnitt variieren. Durch
diese Parameter kann ihr an entsprechenden Stellen mehr oder weniger
Aufmerksamkeit geschenkt werden. So lassen sich Hierarchien zwischen Elementen
erzeugen.
